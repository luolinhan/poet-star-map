## 📖 项目简介

**诗人星图** 是一个专为中小学生设计的古诗词学习平台，通过可视化的方式展示诗人、诗词和朝代信息。

### ✨ 特色功能

- 🌌 **星图可视化** - 每位诗人都是夜空中的一颗星，点击可查看详细信息
- 📖 **完整诗词** - 包含小学、初中、高中课内必背和扩展阅读
- 🏛️ **朝代脉络** - 清晰展示历史朝代沿革，理解诗人时代背景
- 🎨 **颜色分类** - 不同朝代用不同颜色标注，一目了然
- 🔍 **智能筛选** - 按朝代、学段、搜索快速定位诗人
- 📱 **响应式设计** - 支持手机、平板、电脑各种设备

## 🚀 技术栈

- **HTML5** - 语义化结构

## 🛠️ 本地开发

```bash
# 克隆项目
git clone https://github.com/luolinhan/poet-star-map.git
cd poet-star-map

# 使用 Python 简单服务器
python3 -m http.server 8000

# 或使用 Node.js
npx serve .

# 访问 http://localhost:8000
```
